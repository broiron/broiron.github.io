# Hyeongcheol Park — Academic Homepage

A minimal GitHub Pages academic homepage.

## How to deploy

1. Create a GitHub repository named `cchptr204.github.io`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Set the source to the `main` branch and root folder.
5. Visit `https://cchptr204.github.io`.

## What to edit

- Replace `#` links in `index.html` with actual Google Scholar, LinkedIn, paper, project, and code URLs.
- Replace the circular `HP` placeholder with a profile photo if desired.
- Update `assets/files/CV_2026.pdf` whenever the CV changes.
